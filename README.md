1. 프로젝트 디렉토리로 이동

```shell
cd carerr-bot-front
```

2. 리액트 설치

```shell
npm install -g create-react-app
```

3. 리액트 라우터 설치

```shell
npm install react-router-dom
```

4. 리액트 부트스트랩 설치

```shell
npm install react-bootstrap bootstrap
```

5. axios 설치

```shell
npm install axios
```

6. Redux Toolkit 및 react-redux 설치

```shell
npm install @reduxjs/toolkit react-redux
```

7. 리액트 실행

```shell
npm start
```
